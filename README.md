# IGCSE Computer Science Hub (0478)

A structured content hub for CAIE IGCSE Computer Science (0478), covering the 2026–2028 syllabus cycle.

## Purpose

This project aggregates and converts study materials from multiple sources into clean, structured Markdown — the foundation for a published revision website and a set of teacher resources including presentations, workbook questions, and exam preparation materials.

## What's inside

Content is drawn from:
- **Official textbook** — chapter notes and key term definitions
- **PMT Education notes** — detailed structured notes per chapter
- **Craig 'n' Dave presentations** — slide content and summaries for all 108 IGCSE topics
- **Save My Exams** — topic revision notes and flashcards
- **Past exam papers** — Paper 1 & Paper 2 (2021–2025), mapped to syllabus topics and assessment objectives
- **Moshikur.com** — supplementary topic and pseudocode content

## Who it's for

Students studying CAIE IGCSE Computer Science (0478) and teachers looking for ready-to-use resources to support teaching, revision, and exam preparation.
