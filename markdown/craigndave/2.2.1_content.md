### Error checking methods

### The need for error checking
- When data is transmitted, there is always a possibility that data can be lost, gained or changed.
- The most common reasons for data corruption are:
  - Electrical interference (noise)
  - Problems during packet switching
- Common methods of error checking include:
  - Parity checks
  - Checksums
  - Echo checks
  - Automatic Repeat Query (ARQ)
