# 3.4 Network hardware
*The role of a router*

---

### Routers
- Routers come in many different shapes and sizes, but they all essentially do the same thing.
- Receive inbound data and direct  it to a specific device on a network (LAN)
- Join local area networks (LANs) to the internet with a public IP address
- Pass local  data on to an external network (internet)
- Direct traffic (packets) from one router to another using IP addresses

### Routers
- A typical router will have:
  - A cable connecting it to the internet.
  - Several other cables connecting to computers and other devices on the LAN.
  - Wireless connectivity.

### Routers
- A router sends data between networks – it is required to connect a local area network to a wide area network.
- Routers use devices’ IP address to route traffic to other routers.
- Routers can assign IP addresses to devices on a local network.
