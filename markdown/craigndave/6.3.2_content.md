### Artificial intelligence systems
- There are many different types of AI systems.
- The two you need to know about for the exam are expert systems and machine learning.
- Artificial intelligence
- Technique that enables machines to mimic human behaviour.
- Expert systems
- Subset of AI that mimics the decision-making ability of humans using expert knowledge.
- Machine learning
- Subset of AI that uses data to enable machines to be trained and learn from past experiences.

### Expert systems
- Expert systems attempt to mimic human knowledge and experience. In essence, they use a combination of knowledge and inference to solve problems or answer questions in a similar way to a human expert.

### Expert systems
- Expert systems have many advantages.

### What makes up an expert system?
- Expert systems are made up of a number of different components.
- There are four main ones you need to know about.

### User interface
- This is the method by which the user interacts with the expert system.
- It will typically include dialog boxes, prompts and other input methods that present a series of yes/no questions based on responses to previous questions.

### Knowledge base
- The system is initially set up with a repository of facts. It attempts to store all the knowledge about a specific area of expertise, which can be obtained from a variety of resources.
- The knowledge base is essentially a collection of objects and their attributes.

### Rules base
- The rules base is essentially a set of logical statements, much like IF statements.

- In essence, it acts like a search engine. It searches the knowledge base, using the inference rules from the rules base to make decisions.

### Machine learning
- Machine learning is a subset of AI – in other words, it is one way to achieve artificial intelligence.
- Machine learning is the science of training a device or application to perform a task and improve its capabilities – this is achieved by feeding it data, information and scenarios so it can learn over time from its past experiences.
- KEY CHARACTERISTICS
- Possible to make predictions and decisions based on previous scenarios
- Ability to manage and analyse vast volumes of complex data
- Can provide fast, accurate outcomes

### Examples of machine learning
- A great example of machine learning is the automatic tag suggestion feature found on many social media sites such as Facebook™.
- Facebook™ can automatically suggest friends to tag in photos you upload using its proprietary machine learning facial recognition system, DeepFace.
- The system uses a nine-layer neural network with over 120 million connection weights. It was initially trained using over four million images uploaded by users and continued to learn as more images were added.
