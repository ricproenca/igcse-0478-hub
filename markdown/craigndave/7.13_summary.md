### Suitable test data
- Test data needs to include a range of:
  - Normal inputs: Data that should be accepted without causing errors.
  - Abnormal inputs: Data that should be rejected by the program – includes no input when one is expected.
  - Boundary inputs: Data that is the largest/smallest acceptable value as well as the corresponding smallest/largest rejected value.
  - Extreme data: Data that is the largest/smallest acceptable value.
