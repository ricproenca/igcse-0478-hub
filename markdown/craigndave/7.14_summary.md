### number = print("Enter a number")
- total = 1
- for counter = 1 to number
- total = total * counter
- next counter
- output(total)
- What is this program doing?
- It is outputting the factorial of the number entered.
- Factorial 5 = 1*2*3*4*5 = 120
- Output window:
- Enter a number
- 5
- 120
- Note: When the first line of code executes – number = print("Enter a number") – assume the user enters the integer 5.
