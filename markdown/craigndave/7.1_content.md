- Collectively, these phases are known as the software development lifecycle (SDLC).
- At a very simple level, software development involves four identifiable stages.

### Software development lifecycle
- When developing a solution to a problem for a computer system, programmers work through a number of distinct phases, collectively known as the software development lifecycle (SDLC).
- There are four main stages, during which various key tasks need to be performed:
  - Analysis
    - Abstraction
    - Problem decomposition
    - Problem identification
    - Requirements
  - Design
    - Further decomposition
    - Structure diagrams
    - Flowcharts
    - Pseudocode
  - Coding
    - Writing program code
    - Iterative testing
  - Testing
    - Testing program code using test data
