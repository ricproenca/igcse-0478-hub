### Linear search
- Starting from the beginning of a data set, each item is checked in turn to see if it is the one being searched for.
- A linear search:
  - Doesn’t require data to be in order.
  - Works on any type of storage device.
  - Is efficient for small data sets.
  - Is very inefficient for large data sets.

### Linear search
- We perform simple linear searches every day of our lives – e.g., looking for your favourite breakfast cereal on the shelves at the supermarket.

### Linear search
- We have declared a function called linearSearch that takes in two parameters.
- The first is an array of data items called items.
- The second is the item we are going to be searching for in the array, called item_to_find.
- FUNCTION linearSearch(items, item_to_find) RETURNS STRING
- DECLARE index : INTEGER
- DECLARE found : BOOLEAN
- index ß 0
- found ß False
- WHILE (found = FALSE) AND (NOT end of data set) DO
- IF items[index] = item_to_find
- THEN
- found ß True
- ELSE
- index ß index + 1
- ENDIF
- ENDWHILE
- IF found = True
- THEN
- RETURN "Item found at position ", index
- ELSE
- RETURN "Item not found"
- ENDIF
- ENDFUNCTION
