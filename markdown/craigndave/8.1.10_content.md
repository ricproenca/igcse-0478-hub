# 8.1 Programming
*The common Boolean operators*

---

### Boolean operators
- We use Boolean expressions all the time in programming. Typical examples are IF statements and FOR or WHILE loops.
- Boolean expressions typically make use of one of more logical operators and evaluate to either TRUE or FALSE.
- You need to be aware of how the following Boolean operators function.
- AND
- OR
- NOT

### Boolean operators
- IF <BOOLEAN EXPRESSION>
- THEN
- ...
- ENDIF
- WHILE <BOOLEAN EXPRESSION> DO
- ...
- ENDWHILE
- REPEAT
- ...
- UNTIL <BOOLEAN EXPRESSION>
