### Nested structures
- Nesting means putting one statement inside another – this can be achieved with iteration and selection statements, or a mix of both.
- IF GameWon
- THEN
- … instructions here…
- IF Score > HighScore
- THEN
- … instructions here…
- ENDIF
- … instructions here…
- ENDIF
- WHILE NotSolved
- … instructions here…
- FOR i ß 1 TO 5
- … instructions here…
- ENDFOR
- … instructions here…
- ENDWHILE

### Nested statement examples using Cambridge IGCSE pseudocode
- Here we see an example of a FOR loop nested inside another FOR loop.
- This simple program displays all the multiplication tables between 1 and 12.
- FOR Table ß 1 TO 12
- FOR N ß 1 TO 12
- Answer ß Tables * N
- OUTPUT Answer
- NEXT N
- NEXT Table

### Nested statement examples using Cambridge IGCSE pseudocode
- Although you can nest statements within statements to many levels, you will not be expected to write more than three.
- The most complex nest statement in the exam will be something like the following example:
- FOR Table ß 1 TO 12
- FOR N ß 1 TO 12
- IF Table = 12
- THEN
- OUTPUT "Final set of tables"
- ENDIF
- Answer ß Tables * N
- OUTPUT Answer
- NEXT N
- NEXT Table
