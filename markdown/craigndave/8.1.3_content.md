### Beat the Dice
- Beat the Dice is a simple game to help young children learning to compare one- and two-digit numbers:
  - Players roll two dice.
  - Doubles do not count and must be rolled again.
  - The score is the value of the highest and lowest dice combined, highest first (e.g., 5 and 7 = 75).

### #Beat the Dice
- #Import libraries
- import random
- #Constants
- rolls_per_player = 2
- #Initialise varilables
- total = 0
- #Roll the dice
- for rolls in range(rolls_per_player):
- dice1 = 0
- dice2 = 0
- #Handle the possibility of a double throw
- while dice == dice2:
- dice1 = random.randint(1,6)
- dice2 = random.randint(1,6)
- #Output the dice
- print("dice 1:",dice1)
- print("dice 2:",dice2)
- #Cast the dice values from integers to strings ready for changing the order of the dice
- dice1_string = str(dice1)
- dice2_string = str(dice2)
- #Change the order of the dice to the highest value first and join the values together
- if dice1 > dice2:
- roll_value = dice1_string + dice2_string
- else:
- roll_value = dice2_string + dice1_string
- #Cast the value of the dice back to an integers for adding the score to the total
- roll_value = int(roll_value)
- #Ask for the value of the roll
- user_input = int(input("What is the value of the roll?"))
- #Output the value of the dice
- if user_input == roll_value:
- print("You worked it out correctly.")
- else:
- print("No, the value of the roll is:",roll_value)
- #Add the value of the dice to the total
- total = total + roll_value
- #Output the total score of the rolls
- print("The total of the",rolls_per_player,"rolls is:", total)

### Variables and constants
- Variable: A value stored in memory that can change while the program is running – can be an integer, character, string, real/float or Boolean value.
- Constant: A value that is assigned when the program is first written and does not change while it is running.
  - Constants make a program easier to read, as they are usually declared and assigned at the top of the program.
  - They allow programmers to modify a program by changing one value rather than having to change every instance of a value throughout the program, reducing the possibility of errors.
  - If constants are used instead of variables, a compiler can be used to optimise code, making the program run faster.
