### Data types
- Variables and constants hold data. This data comes in a variety of forms, known as its data type.
- You need to be aware of the following five data types for the exam.

### Data types
- Each data type uses a different amount of memory.
- To optimise your programs, you should use correct data types.
- For example, there is no point storing the whole number 22 as a real data type when an integer works and takes up less memory.
- Integer
- Real

### Title
- Integer
  - Positive or negative whole number (e.g., 6)
  - Use: total = total + score
  - Never used for telephone numbers
  - Used only when arithmetic needs to be performed on the data
- Real
  - Number with a decimal part (e.g., 6.5)
  - Sometimes called float (floating point)
  - Use: cost = total + vat
- Char
  - Single alphanumeric character (e.g., “1” or “a”)
  - Used for menu choices
  - Use: choice = input(“Enter your choice:”)
- String
  - Multiple alphanumeric characters (e.g., “Craig”)
  - Used for words and telephone numbers
  - Use: forename = “Craig”
- Boolean
  - Variable that can be in one of two states
  - True or False
  - Use: playerAlive = True
- Most languages support other data types such as dates and pictures.
