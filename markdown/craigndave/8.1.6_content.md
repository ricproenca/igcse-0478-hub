### Iteration (looping)
- Iteration, sometimes called looping, means repeating sections of code.
- A REPEAT loop (also known as a post-condition loop) is used when the required number of iterations is not known ahead of execution because the variable used to determine when the iteration ends is changing within the iteration itself.
- A REPEAT loop will always execute at least once as the check is performed at the end of the loop.
- REPEAT
- OUTPUT "Please enter the password"
- INPUT Password
- UNTIL Password = "Secret"
- A WHILE loop (also known as a pre-condition loop) is an alternative to REPEAT where the condition is checked before entering the loop, this means the loop might execute zero times if the condition is FALSE at the start of the first iteration.
- WHILE Number > 9 DO
- Number ß Number - 1
- ENDWHILE
- Although the language you are learning may not utilise these commands, you still need to be aware of them for the exam.
