### Counting
- Python
- Visual Basic
- NumberOfStudents = NumberOfStudents +1
- NumberOfStudents +=1
- NumberOfStudents = NumberOfStudents +1
- NumberOfStudents +=1
- C#
- Java
- NumberOfStudents = NumberOfStudents +1;
- NumberOfStudents += 1;
- NumberOfStudents++;
- NumberOfStudents = NumberOfStudents +1;
- NumberOfStudents ++;
