### String manipulation
- String manipulation is the act of manipulating, extracting or changing the characters in a string variable.

### String manipulation
- Example 1: An online form
- Regardless of how the user inputs their first and last names, the program should store the names internally with the first letter as a capital and the rest in lower case.
- So, “crAIG SARGEnt” would become “Craig Sargent”.
- This could be achieved by the following steps:
- STEP 1: Extract the first letter from the input and convert it to upper case.
- STEP 2: Extract all the remaining letters and convert them to lower case.
- STEP 3: Join the two parts back together.
- crAIg
- SARGEnt

### String manipulation
- Example 2: Text parsing
- You can input phrases into search engines like “6cm in inches”. The search engine then parses this text string and provides a sensible output.
  - Extract the number 6.
  - Extract the characters “cm”.
  - Extract the characters “inches”.

- e.g., if string was “Hello”, chars would be “l”.
