# Summary: 8.2 Arrays
*The use of arrays*

---

### Arrays
- A static number of related data items stored in sequence within the same memory space.
- Each data item has the same (homogeneous) data type.
- A particular data item (element) is found using its index.
- Indexes usually start at 0 for the first data item (known as zero-indexed) but can also start from 1.
- Arrays may be single- or multi-dimensional.
- Visualise dimensions as a column (one-dimensional) or table (two-dimensional).
- Note: [1][0] could also be 112 depending on how you choose to store the data. It doesn’t matter whether you visualise rows or columns first, providing you are consistent in your program. Read the question carefully in an exam so you can be sure which way around the array is being stored.
- names[2] = “Sam”
- names[1][0] = “Dave”
