### Arrays and iteration
- An array can contain a lot of separate data items, and it is often necessary to travel through all these elements:
- Populating an entire array
- Traversing an array to find/update an item
- Outputting the contents of an array
- Luckily, there is an ideal programming construct to support this – iteration, or FOR loops.

### Arrays and iteration
- Here we are using a FOR loop to populate the test scores of ten students into a one-dimensional array.
- DECLARE TestScores : ARRAY[0:9] OF INTEGER
- FOR Counter ← 0 TO 9
- OUTPUT "Enter the next test score: "
- INPUT TestScores[Counter]
- NEXT Counter
- Note: Each element is referenced by its index number. IGCSE pseudocode tends to use [1] as the first index, but many languages use [0].
- Output window:
- Enter the next test score: 47
- Enter the next test score: 56
- Enter the next test score: 12
- ...

### Arrays and iteration
- Here we are using a FOR loop to output the contents of the TestScores array.
- DECLARE TestScores : ARRAY[0:9] OF INTEGER
- FOR Counter ← 0 TO 9
- OUTPUT "Test score ", Counter + 1, ": ", TestScores[Counter]
- NEXT Counter
- Output window:
- Test score 1: 47
- Test score 2: 56
- Test score 3: 12
- Test score 4: 19
- ...

### Arrays and iteration
- Here we are using a nested FOR loop to populate the lowest and highest marks of ten students into a two-dimensional array.
- DECLARE TestScores : ARRAY[0:9, 0:1] OF INTEGER
- FOR CounterX ← 0 TO 9
- OUTPUT "Student ", CounterX +1
- OUTPUT "Enter low score followed by high score"
- FOR CounterY ← 0 TO 1
- OUPUT "Input score: "
- INPUT TestScores[CounterX, Counter Y]
- NEXT CounterY
- NEXT CounterX
- Output window:
- Student 1
- Enter low score followed by high score
- Input score: 13
- Input score: 89
- Student 2
- Enter low score followed by high score
- Input score: 19
- Input score: 73

### Arrays and iteration
- Here we are using a single FOR loop to output the contents of the TestScores array.
- DECLARE TestScores : ARRAY[0:9, 0:1] OF INTEGER
- OUTPUT "No. Low  High“
- OUTPUT "============="
- FOR Counter ← 0 TO 9
- OUTPUT Counter + 1, "   ",TestScores[Counter,0], "   ", TestScores[Counter,1]
- NEXT Counter
- Output window:
- No. Low  High
- =============
- 1   13   89
- 2   19   73
- 3   11   95
- 4   13   37
- ...
