### Single-table databases
- Databases are structured, persistent collections of data.
- Databases come in two forms – single-table, flat file databases and relational databases. You are only  required to know about single-table databases.
- Flat file databases
  - Consist of separate tables.
  - Each table is made up of a number of records.
  - Each record has a number of separate fields.
  - Fields can be one of several basic data types – text/alphanumeric, character, Boolean, integer, real, date/time.
  - Each table in a database should have a primary key. The primary key field must be a field that can uniquely identify each record in the table.
  - Database table fields can be assigned various validation rules – length check, format check, range check, presence check, custom rule, etc.
