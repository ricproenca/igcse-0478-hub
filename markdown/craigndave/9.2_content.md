### SQL: Structured Query Language
- Information in a database is stored in records. Each record can contain a number of fields.
- We can manipulate the data in these fields using a programming language called Structured Query Language (SQL).

### SQL: Structured Query Language
- Structured Query Language (SQL) statements supports a wide range of commands for manipulating a database.
- The one you need to familiar with for the IGCSE exam is the SELECT statement.

### SQL: Structured Query Language
- Structured Query Language (SQL) statements follow a strict format.
- The format for the SELECT statement is as follows:
- SELECT…
- FROM…
- WHERE…
- ORDER BY…

### SQL: Structured Query Language
- SQL is used to create, delete, modify and manipulate records in a database. Basic commands include:
  - SELECT which fields to be returned.
  - FROM which table – databases can have more than one table, each with their own unique name.
  - WHERE records meet a condition.
  - LIKE can be used as a wildcard.
  - ORDER BY: Choose how the data is organised when it is retrieved (e.g., ORDER BY name).
    - ORDER BY <field> ASC: Order by the selected field in ascending order.
    - ORDER BY <field> DESC: Order by the selected field in descending order.
  - SELECT SUM <field>: Specifies fields for calculation – should be integer or real.
  - SELECT COUNT <field>: Counts fields that meet the specified criteria.
