# Chapter 08 Definitions

*Topic 8: Programming*


## 1D Array

A single list of items of the same data type.


## 2D Array

An array of arrays, similar to a table or grid.


## Addition

An arithmetic operation.


## AND

A Boolean operation.


## Arithmetic Operation

Operators that perform calculations (e.g., +, -, *, /).


## Arrays

A data structure that stores a collection of items, typically of the same data type, accessed using an index.


## Assignment

Setting or updating a value in a variable.


## Boolean

A data type representing true or false values.


## Character

A data type representing a single character.


## Condition-controlled loop

A loop which repeats code until a condition is met. Education is licensed under CC BY-NC-ND


## Constant

A value that does not change while the program runs. Often given fully uppercase identifiers. Variable/Constant Declaration: Creates a variable/constant (including the value’s identifier, a label used to uniquely identify the memory location) to store data.


## Count-controlled loop

A loop which repeats code a certain number of times (for loop).


## Counting

Keeping track of how many times an event has occurred.


## Data Type

Defines the kind of data a variable or constant can hold. It tells the program how the data will be stored, processed, and displayed.


## File Handling

Operations which allow for the reading of and writing to an external file containing permanently saved data.


## Function

A subroutine which performs an action and returns a value.


## Global Variables

A variable that exists in memory throughout the execution of a program and can be accessed by any part of a program.


## Input

When a program receives data from a user.


## Integer

A data type representing whole numbers.


## Integer Division

An arithmetic operation that produces the integer quotient (whole part) of a division.


## Iteration

Repeating a set of instructions (e.g., WHILE, FOR loops).


## Length

A string operation which returns the length of the string.


## Local Variables

A variable that exists in memory only during the execution of a subroutine and cannot be accessed outside of that subroutine.


## Logical Operation

AND, OR, NOT used in logical conditions.


## Lower

A string operation that can be used to put the string in full-lowercase.


## Multiplication

An arithmetic operation.


## Nested Statement

Iteration and selection statements which are nested (indented) within each other to provide more complex decision-making or program flow.


## NOT

A Boolean operation.


## OR

A Boolean operation.


## Output

Displaying data to the user.


## Parameter

A variable that allows data to be passed into a subroutine.


## Post-condition loop

The condition is checked at the end of the loop (do…until loop).


## Pre-condition loop

The condition is checked before the loop (while loop).


## Procedure

A subroutine which performs an action and does not return a value.


## Random

A library routine that can be used to generate random numbers.


## Reading

Reading from an external file.


## Real (Float)

A data type representing numbers with fractional parts.


## Real Division

An arithmetic operation that produces a real number result.


## Relational Operation

Operators that compare values (e.g., ==, !=, >).


## Remainder (Modular Arithmetic)

The value left over when one integer is divided by another (e.g., 11 MOD 2 = 1). Equal To (==): A relational operation used to check if two values are the same. Not Equal To (< > or !=): A relational operation used to check if two values aren’t the same. Less Than (<): A relational operation used to check if one value is less than another. Greater Than (>): A relational operation used to check if one value is greater than another. Less Than Or Equal To (<=): A relational operation used to check if one value is less than, or equal to, another. Greater Than Or Equal To (>=): A relational operation used to check if one value is greater than, or equal to, another.


## ROUND

A library routine that can be used to round numbers.


## Selection

Using conditions to control the flow of a program (e.g., IF statements).


## Sequence

Executing instructions one after the other, in the order they were written.


## String

A data type representing a sequence of characters.


## String Handling

Operations which can be performed on strings.


## Substring

A string operation which can be used to split a string or copy a section of it.


## Subtraction

An arithmetic operation.


## Totalling

Keeping a running sum of numbers throughout a program or loop.


## Upper

A string operation that can be used to put the string in full-uppercase.


## Variable

A value stored in memory that can change during run-time.


## Writing

Writing to an external file.
