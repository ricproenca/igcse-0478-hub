# Cambridge IGCSE Computer Science 0478 - Syllabus

## Assessment Objectives

| Code | Description |
|------|-------------|
| **AO1** | Demonstrate knowledge and understanding of the principles and concepts of computer science |
| **AO2** | Apply knowledge and understanding of the principles and concepts of computer science to a given context, including the analysis and design of computational or programming problems |
| **AO3** | Provide solutions to problems by evaluating computer systems, making reasoned judgements, and presenting conclusions |

### Assessment Objectives Weighting

| Assessment Objective | Weighting (IGCSE %) |
|---------------------|---------------------|
| AO1 | 40% |
| AO2 | 40% |

---

## Subject Content

### Paper 1: Computer Systems (Topics 1-6)

---

## 1. Data Representation

### 1.1 Number Systems
- Number systems (denary, binary, hexadecimal)
- Conversions between denary, binary, and hexadecimal
- Binary addition and overflow errors
- Logical binary shifts (left and right shifts)
- Two's complement representation

### 1.2 Text, Sound and Images
- Text representation (ASCII and Unicode character sets)
- Sound representation (sample rate and sample resolution)
- Image representation (resolution and colour depth)

### 1.3 Data Storage and Compression
- Data storage measurement (bit, nibble, byte, KiB, MiB, GiB, TiB, PiB, EiB)
- File size calculations for images and sounds
- Data compression (lossless and lossy methods)

---

## 2. Data Transmission

### 2.1 Types and Methods of Data Transmission
- Packet switching (structure of packets: header, payload, trailer)
- Data transmission methods (serial, parallel, simplex, half-duplex, full-duplex)
- USB interface

### 2.2 Methods of Error Detection
- Parity check (odd and even)
- Checksum
- Echo check
- Check digit (ISBN, bar codes)
- Automatic Repeat Query (ARQ)

### 2.3 Encryption
- Symmetric encryption
- Asymmetric encryption (public and private keys)

---

## 3. Hardware

### 3.1 Computer Architecture
- Role of the CPU and microprocessor
- Von Neumann architecture components (ALU, CU, registers, buses)
- Fetch-Decode-Execute (FDE) cycle
- CPU performance factors (cores, cache, clock)
- Instruction sets
- Embedded systems

### 3.2 Input and Output Devices
- Input devices (barcode scanner, digital camera, keyboard, microphone, optical mouse, QR code scanner, touch screen, 2D/3D scanners)
- Output devices (actuator, DLP projector, inkjet/laser printer, LED/LCD screen, speaker, 3D printer)
- Sensors (acoustic, accelerometer, flow, gas, humidity, infrared, level, light, magnetic field, moisture, pH, pressure, proximity, temperature)

### 3.3 Data Storage
- Primary storage (RAM, ROM)
- Secondary storage
- Storage types (magnetic, optical, solid-state/flash memory)
- Virtual memory
- Cloud storage

### 3.4 Network Hardware
- Network Interface Card (NIC)
- MAC address
- IP address (IPv4, IPv6)
- Router

---

## 4. Software

### 4.1 Types of Software and Interrupts
- System software vs application software
- Operating system functions
- Hardware, firmware, and operating system hierarchy
- Interrupts (generation, handling, interrupt service routine)

### 4.2 Types of Programming Language, Translators and IDEs
- High-level languages vs low-level languages
- Assembly language and assembler
- Compiler vs interpreter
- Integrated Development Environment (IDE) functions

---

## 5. The Internet and its Uses

### 5.1 The Internet and the World Wide Web
- Internet vs World Wide Web
- URL
- HTTP and HTTPS
- Web browser functions
- How web pages are retrieved and displayed
- Cookies (session and persistent)

### 5.2 Digital Currency
- Digital currency concept
- Blockchain technology

### 5.3 Cyber Security
- Cyber security threats (brute-force attack, data interception, DDoS, hacking, malware, pharming, phishing, social engineering)
- Security solutions (access levels, anti-malware, authentication, firewalls, SSL, etc.)

---

## 6. Automated and Emerging Technologies

### 6.1 Automated Systems
- Sensors, microprocessors, and actuators in automated systems

### 6.2 Robotics
- Robotics definition and characteristics
- Robot components and capabilities

### 6.3 Artificial Intelligence
- AI definition
- AI characteristics (data, rules, reasoning, learning)
- Expert systems
- Machine learning

---

### Paper 2: Algorithms, Programming and Logic (Topics 7-10)

---

## 7. Algorithm Design and Problem-Solving

### 7.1 Program Development Life Cycle
- Analysis (abstraction, decomposition)
- Design (structure diagrams, flowcharts, pseudocode)
- Coding and iterative testing
- Testing

### 7.2 Problem Decomposition
- Sub-systems decomposition
- Inputs, processes, outputs, storage

### 7.3 Algorithm Design Methods
- Structure diagrams
- Flowcharts
- Pseudocode

### 7.4 Standard Methods of Solution
- Linear search
- Bubble sort
- Totalling
- Counting
- Finding maximum, minimum, and average values

### 7.5 Validation and Verification
- Validation checks (range, length, type, presence, format, check digit)
- Verification checks (visual check, double entry check)

### 7.6 Testing
- Test data types (normal, abnormal, extreme, boundary)

### 7.7 Trace Tables
- Tracing algorithm execution

### 7.8 Error Identification
- Identifying and correcting errors in algorithms

---

## 8. Programming

### 8.1 Programming Concepts
- Variables and constants
- Data types (integer, real, char, string, Boolean)
- Input and output
- Sequence
- Selection (IF statements, CASE statements)
- Iteration (count-controlled, pre-condition, post-condition)
- Totalling and counting
- String handling (length, substring, upper, lower)
- Arithmetic, logical, and Boolean operators
- Nested statements
- Procedures and functions (parameters, local/global variables)
- Library routines
- Maintainable programs (meaningful identifiers, comments)

### 8.2 Arrays
- 1D and 2D arrays
- Array operations using iteration

### 8.3 File Handling
- Purpose of file storage
- Opening, closing, reading, and writing files

---

## 9. Databases

### 9.1 Database Design
- Single-table database structure (fields, records)
- Data types (text, character, Boolean, integer, real, date/time)
- Primary key

### 9.2 SQL Queries
- SELECT, FROM, WHERE, ORDER BY
- SUM, COUNT

---

## 10. Boolean Logic

### 10.1 Logic Gates
- Logic gate symbols (NOT, AND, OR, NAND, NOR, XOR)
- Gate functions

### 10.2 Logic Circuits
- Creating circuits from problem statements, expressions, and truth tables
- Truth tables for logic circuits
- Writing logic expressions from circuits and truth tables

---
