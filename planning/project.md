# Project Overview: igcse-0478-hub

CAIE IGCSE Computer Science (0478) study hub — a collection of structured markdown content converted from multiple source materials.

---

## `markdown/` — Output Folders

### `markdown/textbook/`
Chapter-by-chapter markdown conversions of the official CAIE IGCSE Computer Science textbook (Chapters 1–10). Each chapter includes extracted images saved under `images/<chapter>/`.

### `markdown/notes/`
PMT Education chapter notes — detailed structured notes for each of the 10 syllabus chapters, with sections, subsections, tables, and diagrams.

### `markdown/definitions/`
Per-chapter definition files extracted from the textbook. Each file covers the "Key terms" for one chapter, formatted as `## term` followed by its definition.

### `markdown/papers/`
Past exam paper files (Paper 1 and Paper 2, 2021–2025) structured as markdown tables mapping questions to syllabus topics, difficulty, marks, and assessment objectives.

### `markdown/presentations/`
Presentation-style notes derived from Craig 'n' Dave IGCSE video slides — currently not populated.

### `markdown/craigndave/`
Content and summary markdown files extracted from all 108 Craig 'n' Dave PPTX presentations. Files are named by topic slug (e.g. `1.1.1_content.md`, `1.1.1_summary.md`). "Going beyond the IGCSE specification" slides are excluded. Covers chapters 1–10.

### `markdown/savemyexams/`
Markdown conversions of Save My Exams topic revision PDFs, one file per topic. Includes extracted diagrams and figures saved under `images/`.

### `markdown/savemyexams_flashcards/`
Markdown conversions of Save My Exams flashcard PDFs, one file per topic. Flashcard content is image-heavy; diagrams are saved under `images/`.

### `markdown/moshikur_igcse/`
Markdown content scraped from moshikur.com covering IGCSE Computer Science topics. Organised by syllabus topic.

### `markdown/moshikur_pseudocode/`
Markdown content scraped from moshikur.com covering pseudocode concepts and programming constructs. Organised by topic.
