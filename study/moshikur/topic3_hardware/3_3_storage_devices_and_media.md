# 3.3.1 Primary Storage

## 3.3.1.1 Understand what is meant by primary storage.

---

[![Illustration depicting primary storage components: RAM representing temporary data, and ROM representing permanent data.](images/topic3_hardware/3_3_storage_devices_and_media/img_001.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-56.png?ssl=1)

# What is Primary Storage?

Primary storage, also called **main memory**, is the **fast memory** that a computer uses to store data and instructions that are currently needed by the CPU.

It includes both **temporary memory (RAM)** and **permanent memory (ROM)**. Unlike secondary storage, primary storage is directly accessed by the CPU, which makes it much faster.

---

### Key Features of Primary Storage

- **Direct access** by the CPU for fast processing.
- Includes **volatile memory** (RAM) and **non-volatile memory** (ROM).
- Holds data and instructions that are in use.
- Essential for running the operating system and applications.

---

### Why is Primary Storage Important?

- Without primary storage, the CPU would have no place to quickly read and write data.
- It ensures smooth operation of programs.
- It provides both **temporary storage for active data** (RAM) and **permanent storage for essential instructions** (ROM).

---

### Examples of Primary Storage

- **RAM (Random Access Memory):** Temporary storage, cleared when power is off.
- **ROM (Read-Only Memory):** Permanent storage, contains startup instructions for the computer.

---

## 3.3.1.2 Random Access Memory

---

### What is RAM?

RAM (Random Access Memory) is a type of **primary storage** that temporarily stores data and instructions while a computer is running. It is **volatile memory**, meaning all contents are lost when the power is switched off.

---

### Key Features

- **Temporary storage** – used only while the computer is on.
- **Volatile** – loses data when power is off.
- **Fast access speed** – much quicker than secondary storage.
- Stores both **data** and **program instructions** currently in use.
- Can be **read from and written to**.

---

### Purpose of RAM

- Holds the **operating system**, applications, and data that the CPU needs while the computer is running.
- Allows the computer to **multi-task** by keeping several programs open at once.
- Provides the CPU with **quick access** to active data, which improves performance.

---

### Why is RAM Important?

- The more RAM a computer has, the more programs it can run smoothly at the same time.
- If the RAM is full, the computer may use **virtual memory**, which is slower.
- It plays a key role in ensuring **fast and efficient processing**.

---

### Example Uses of RAM

- Running applications like word processors, web browsers, or games.
- Holding data files temporarily while being edited.
- Supporting background tasks such as system updates.

---

## 3.3.1.3 ROM (Read-Only Memory)

---

### What is ROM?

ROM (Read-Only Memory) is a type of **primary storage** that permanently stores essential data and instructions. It is **non-volatile memory**, which means its contents are **not lost when the power is switched off**.

---

### Key Features

- **Permanent storage** – data is written during manufacture and cannot normally be changed.
- **Non-volatile** – retains contents even when the computer is turned off.
- Stores **essential instructions** needed for the computer to operate.
- **Read-only** – data can be read but not usually altered.

---

### Purpose of ROM

- Stores the **BIOS (Basic Input/Output System)** or firmware, which is needed to start up the computer.
- Provides the instructions for performing **hardware checks** when the computer is switched on.
- Loads the operating system from secondary storage into RAM.

---

### Why is ROM Important?

- Without ROM, the computer would not know how to **start up**.
- It ensures that the **boot-up process** is always available, even if power is lost.
- Provides a **reliable and permanent** set of startup instructions.

---

### Example Uses of ROM

- Storing the BIOS in desktop and laptop computers.
- Embedded systems (e.g., washing machines, microwaves, calculators).
- Game consoles and other devices where permanent instructions are needed.

# 3.3.2 Secondary Storage

### 3.3.2.1 Understand What is Meant by Secondary Storage

---

[![Illustration showing types of secondary storage devices including a hard disk drive, a solid-state drive, and an optical disc.](images/topic3_hardware/3_3_storage_devices_and_media/img_002.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-57.png?ssl=1)

## What is Secondary Storage?

Secondary storage is a type of **non-volatile storage** that holds data and programs **permanently**, even when the computer is switched off. Unlike primary storage, it is not directly accessed by the CPU but provides long-term storage for files, applications, and the operating system.

---

### Key Features of Secondary Storage

- **Non-volatile** – data is not lost when the power is off.
- Used for **long-term storage** of data and software.
- Much **slower** than primary storage (RAM and ROM).
- Available in many types and sizes, such as magnetic, optical, and solid-state.

---

### Purpose of Secondary Storage

- Stores the **operating system** permanently.
- Holds **applications and programs** for later use.
- Stores **user data**, such as documents, images, videos, and music.
- Provides a place for **backups and archives**.

---

### Examples of Secondary Storage

- **Magnetic storage** – Hard Disk Drives (HDDs).
- **Optical storage** – CDs, DVDs, Blu-ray discs.
- **Solid-state storage** – SSDs, USB flash drives, memory cards.

---

## 3.3.2.2 Describe the Need for Secondary Storage

---

### Why Do We Need Secondary Storage?

Secondary storage is essential because **primary storage (RAM and ROM)** alone is not enough for a computer system. RAM is temporary and volatile, and ROM only contains fixed instructions. Therefore, computers require a way to **store data and programs permanently**.

---

### Key Reasons for Needing Secondary Storage

1. **Permanent Storage of Data**
   - Keeps files, programs, and the operating system safe when the computer is switched off.
2. **Large Storage Capacity**
   - Secondary storage provides much larger capacity than RAM or ROM, making it suitable for holding applications, media, and backups.
3. **Backup and Security**
   - Data can be stored safely for recovery in case of system crashes or corruption.
4. **Portability and Sharing**
   - Devices such as USB flash drives and optical discs allow users to carry and transfer data easily.
5. **Cost Efficiency**
   - Secondary storage is cheaper per gigabyte compared to primary storage.

---

### Examples of Secondary Storage Use

- Storing the **operating system** and application software.
- Saving **documents, photos, and videos** for long-term use.
- Backing up important files to prevent **data loss**.
- Distributing software or media on CDs, DVDs, or Blu-ray discs.

---

### 3.3.3.1 Magnetic Storage: Operation and Examples

---

[![Illustration of a hard disk drive labeled 'MAGNETIC STORAGE', depicting a circular magnetic disk with a reading head.](images/topic3_hardware/3_3_storage_devices_and_media/img_003.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-58.png?ssl=1)

### What is Magnetic Storage?

Magnetic storage is a type of **secondary storage** that uses magnetic fields to record data on a surface. It is one of the oldest and most widely used forms of storage for computers.

---

### How Does Magnetic Storage Work?

1. The storage device has a disk coated with a magnetic material.
2. A **read/write head** moves over the disk surface.
3. Data is written by magnetising tiny spots on the surface in different directions (representing binary 0s and 1s).
4. To read data, the head detects the magnetic fields of these spots and converts them back into electrical signals.

[![Diagram illustrating how magnetic storage works, showing a disk coated with a magnetic material and a read/write head that moves over the disk surface. It explains data writing by magnetising spots representing binary 0s and 1s, and data reading by detecting magnetic fields.](images/topic3_hardware/3_3_storage_devices_and_media/img_004.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/10/image-12.png?ssl=1)

---

### Examples of Magnetic Storage

- **Hard Disk Drive (HDD):** The most common form of magnetic storage in computers.
- **Magnetic Tapes:** Used for large backups and archiving data.
- **Floppy Disks (outdated):** Small portable magnetic storage once used for transferring files.

---

### Advantages of Magnetic Storage

- **Large capacity** – can store terabytes of data.
- **Low cost per gigabyte** – cheaper than solid-state drives.
- **Reliable for long-term storage**.

---

### Disadvantages of Magnetic Storage

- **Slower** than solid-state storage.
- Contains **moving parts**, making it more fragile and prone to damage.
- Generates **heat and noise** during operation.
- Less portable compared to smaller storage devices like USB flash drives.

---

### Common Uses of Magnetic Storage

- Desktop computers and laptops (HDDs).
- Backup systems (magnetic tapes).
- Data servers requiring high-capacity storage.

## 3.3.3.2 Optical Storage: Operation and Examples

---

[![Illustration of optical storage technology showcasing a disc and a laser, representing how data is read and written.](images/topic3_hardware/3_3_storage_devices_and_media/img_005.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-59.png?ssl=1)

### What is Optical Storage?

Optical storage is a type of **secondary storage** that uses **lasers** to read and write data on a reflective surface, such as a disc. It is commonly used for distributing media and software.

---

### How Does Optical Storage Work?

1. Data is stored on the disc in the form of **pits and lands** (tiny bumps and flat areas).
2. A **laser beam** shines onto the surface of the disc.
3. Light reflects differently from pits and lands.
4. A sensor detects these differences and converts them into binary data (0s and 1s).

[![](images/topic3_hardware/3_3_storage_devices_and_media/img_006.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/10/image-13.png?ssl=1)

---

### Examples of Optical Storage

- **CD (Compact Disc):** Holds about 700 MB of data.
- **DVD (Digital Versatile Disc):** Holds 4.7 GB (single-layer) or 8.5 GB (dual-layer).
- **Blu-ray Disc:** Holds up to 25 GB (single-layer) or 50 GB (dual-layer).

---

### Advantages of Optical Storage

- **Portable** – discs are lightweight and easy to carry.
- **Cheap to produce** – especially for mass distribution of media/software.
- **Good for sharing data** like movies, music, and software.

---

### Disadvantages of Optical Storage

- **Limited capacity** compared to HDDs and SSDs.
- **Slower access speed** than other storage types.
- Discs can be **scratched or damaged** easily.
- Becoming **less common** as cloud storage and USB drives are more popular.

---

### Common Uses of Optical Storage

- **Movies and music distribution** (CDs, DVDs, Blu-ray).
- **Software installation discs.**
- **Game consoles** that use Blu-ray discs.
- **Backup storage** (though less common today).

---

## 3.3.3.3 Solid-State Storage: Operation and Examples

---

[![Illustration of a solid-state drive (SSD) labeled 'SOLID-STATE STORAGE' with a lightning bolt symbol, representing fast data storage technology.](images/topic3_hardware/3_3_storage_devices_and_media/img_007.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-60.png?ssl=1)

### What is Solid-State Storage?

Solid-state storage is a type of **secondary storage** that uses **flash memory chips** to store data. Unlike magnetic or optical storage, it has **no moving parts**, making it faster, more reliable, and more durable.

---

### How Does Solid-State Storage Work?

1. Solid-state storage is **non-volatile**. This means that the data is not lost when the power is turned off.
2. Solid-state storage is made of **transistors** that are laid out in a **grid**.
3. Gates are used to control the flow of **electrons** through the transistors. This changes the data in the transistors from 1 to 0, or from 0 to 1.
4. Data can be read or written by sending electrical signals to the cells.
5. Because there are no moving parts, access is almost **instantaneous**.

[![](images/topic3_hardware/3_3_storage_devices_and_media/img_008.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/10/image-14.png?ssl=1)

---

### Examples of Solid-State Storage

- **Solid-State Drives (SSDs):** High-speed storage for computers.
- **USB Flash Drives:** Portable storage devices.
- **Memory Cards (SD cards, microSD):** Used in cameras, phones, and tablets.

---

### Advantages of Solid-State Storage

- **Very fast** read/write speeds compared to HDDs and optical storage.
- **Durable** – no moving parts, less risk of damage.
- **Portable and lightweight.**
- **Silent operation** – no noise.
- **Low power consumption.**

---

### Disadvantages of Solid-State Storage

- **More expensive** per gigabyte than magnetic storage.
- Limited **write cycles** – performance can degrade over time (though modern devices last long).
- Lower maximum storage capacity compared to some HDDs.

---

### Common Uses of Solid-State Storage

- **Laptops and desktops** – SSDs for faster boot-up and application loading.
- **Portable storage** – USB drives for transferring data.
- **Smartphones and tablets** – built-in flash memory.
- **Cameras and drones** – SD cards for image and video storage.

# 3.3.4 Virtual Memory

---

[![Illustration explaining virtual memory, showing data transfer between RAM and disk.](images/topic3_hardware/3_3_storage_devices_and_media/img_009.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-61.png?ssl=1)

### What is Virtual Memory?

Virtual memory is a section of **secondary storage (usually a hard drive or SSD)** that is used by the computer as if it were **extra RAM**.

It acts as an extension of primary storage when the computer’s physical RAM is full, allowing more programs to run at the same time.

---

### Key Features of Virtual Memory

- Located on the **hard drive or SSD**.
- Used when **RAM is full**.
- Allows the system to run larger programs or multiple applications at once.
- Slower than real RAM because secondary storage has a lower access speed.

---

### Why Virtual Memory is Needed

- **RAM is limited** in size, and running many programs may require more memory than is physically available.
- Virtual memory prevents the system from crashing by temporarily moving unused data from RAM to the hard drive.
- It gives the **illusion of more RAM**, helping with multitasking.

---

### Example of Virtual Memory in Use

- A user has several applications open: a web browser, a word processor, and a video editor.
- RAM fills up with active tasks.
- Older or less-used data is temporarily moved to the hard drive as **virtual memory**.
- When needed again, this data is swapped back into RAM.

---

### How is Virtual Memory Created?

- When **RAM becomes full**, the operating system sets aside a portion of the **hard drive (HDD) or SSD**.
- This reserved section acts like **extra RAM** and is called **virtual memory**.
- The space used is often called a **swap file** or **page file**.

---

### How is Virtual Memory Used?

1. Programs and data that are **not immediately needed** are moved from RAM to the virtual memory space.
2. This frees up space in RAM for active tasks.
3. If the data in virtual memory is required again, it is **swapped back into RAM**.
4. This process allows the computer to keep running multiple applications, even with limited physical RAM.

---

### Key Points to Remember

- Virtual memory helps prevent the computer from **running out of memory**.
- It is **slower** than real RAM because hard drives and SSDs are slower to access.
- Using SSDs for virtual memory is faster than HDDs but can reduce SSD lifespan with heavy use.

---

### Why Do Computers Need Virtual Memory?

Virtual memory is necessary because **RAM is limited**. When the computer runs many applications or very large programs, the available RAM may not be enough to store all the active data and instructions.

Without virtual memory, the computer would run out of memory and either slow down severely or crash.

---

### Reasons Virtual Memory is Necessary

1. **Supports Multitasking**
   - Allows multiple programs to run at once, even if RAM is too small.
2. **Handles Large Applications**
   - Some programs (e.g., video editing, games) require more memory than available RAM.
   - Virtual memory provides extra space so they can run.
3. **Prevents System Crashes**
   - Instead of shutting down when RAM is full, the system moves unused data to virtual memory.
4. **Gives Flexibility**
   - Provides the **illusion of more RAM**, making the system more capable.

---

### Limitation of Virtual Memory

- Virtual memory is much **slower than RAM** because it uses the hard drive or SSD.
- Heavy reliance on virtual memory can cause **performance issues** like lag or freezing.

# 3.3.5 Cloud Storage

---

[![An illustration of cloud storage featuring a cloud icon with an upward arrow, connected by lines to a desktop computer, a smartphone, and a tablet, symbolizing data transfer.](images/topic3_hardware/3_3_storage_devices_and_media/img_010.png)](https://i0.wp.com/moshikur.com/wp-content/uploads/2025/09/image-62.png?ssl=1)

### What is Cloud Storage?

Cloud storage is a type of **secondary storage** where data is stored on **remote servers** instead of being kept only on the user’s own computer or device.

The data is managed by a storage provider and is accessed through the **internet**, meaning users can upload, store, and retrieve files from anywhere with an internet connection.

---

### Key Features of Cloud Storage

- **Online access** – data can be reached from multiple devices.
- **Remote servers** – storage is maintained off-site, not on the user’s local machine.
- **Scalable capacity** – users can expand or reduce storage space as needed.
- **Automatic backup** – files can be synchronised and updated across devices.

---

### Examples of What Can Be Stored in the Cloud

- Text documents and presentations.
- Photos, audio files, and videos.
- Backups of software and system data.

---

### How Does Cloud Storage Work?

Cloud storage works by saving data on **remote servers** that are connected to the internet, instead of storing it only on a user’s own computer or device.

1. A user uploads files through a device (such as a computer, tablet, or phone).
2. The files are transferred over the internet to a remote storage system.
3. The data is stored in secure data centres managed by the storage provider.
4. When the user wants to access the files, they connect to the internet and download or view them.
5. The system can also **synchronise files** so that updates made on one device appear on all other devices linked to the account.

---

### Key Features of the Process

- **Remote access** – files can be reached from anywhere with internet.
- **Redundancy** – copies of files are often stored on multiple servers to prevent loss.
- **Scalability** – users can increase storage space as needed without buying new hardware.
- **Collaboration** – more than one person can access and update the same file at the same time.

---

### Advantages of Cloud Storage

1. **Accessibility from Anywhere**
   - Files can be accessed from any device with an internet connection.
   - Useful for working or studying in different locations.
2. **Easy Sharing and Collaboration**
   - Multiple users can view or edit the same file at the same time.
   - Makes teamwork easier without the need for physical file transfer.
3. **Automatic Backup and Recovery**
   - Files stored in the cloud are not lost if the local device is damaged, lost, or stolen.
   - Protects against data loss.
4. **Saves Local Storage Space**
   - Reduces the need to buy large storage devices.
   - Frees up memory on local computers and phones.
5. **Scalability**
   - Storage size can be increased or decreased depending on the user’s needs.
   - No need to buy new hardware to expand capacity.
6. **Cost Efficiency**
   - Reduces the cost of purchasing and maintaining physical storage devices.

---

### Disadvantages of Cloud Storage

1. **Requires Internet Connection**
   - Files cannot be accessed without an internet connection.
   - Slow connections can make uploading and downloading files difficult.
2. **Security Risks**
   - Data stored on remote servers could be targeted by hackers.
   - Sensitive information may be at risk if not properly protected.
3. **Ongoing Costs**
   - Many providers require users to pay regularly for storage space.
   - Costs can increase if large amounts of data are stored.
4. **Privacy Concerns**
   - Data is stored on third-party servers, meaning the user has less control.
   - There may be concerns about who can access or monitor the files.
5. **Data Transfer Limits**
   - Uploading and downloading large files can take time.
   - Some systems may restrict file size or speed.
6. **Dependence on Provider**
   - If the storage provider has technical issues, the data may be temporarily unavailable.
   - Users depend on the provider’s reliability.

---

### Cloud Storage

- **Location:** Data is stored on remote servers accessed via the internet.
- **Accessibility:** Can be accessed from any device with internet.
- **Capacity:** Easily scalable; users can increase or reduce storage as needed.
- **Cost:** May involve ongoing subscription fees.
- **Security:** Risk of hacking or privacy concerns since data is stored off-site.
- **Backup:** Files are usually backed up automatically by the provider.

---

### Local Storage

- **Location:** Data is stored on physical devices such as hard drives, SSDs, or USB drives.
- **Accessibility:** Only accessible from the device where the data is stored (unless manually shared).
- **Capacity:** Limited by the size of the device; to expand, new hardware must be purchased.
- **Cost:** One-time purchase cost of the device; no ongoing subscription fees.
- **Security:** Data is fully controlled by the user; less risk of external hacking, but higher risk if the device is lost, stolen, or damaged.
- **Backup:** Users must manage their own backups.

---

### Summary Comparison

| Feature | Cloud Storage | Local Storage |
| --- | --- | --- |
| Location | Remote servers (online) | Physical device (offline) |
| Accessibility | Anywhere with internet | Only on the device itself |
| Capacity | Scalable | Limited by device size |
| Cost | Ongoing fees possible | One-time purchase |
| Security | Risk of hacking, privacy concerns | Risk of physical damage or loss |
| Backup | Automatic, provider-managed | User must manage manually |

# Exam Style Questions

### **Task 1: Identify Types of Storage**

Identify and categorize the following storage devices as magnetic, optical, or solid-state: hard disk drive (HDD), CD, DVD, USB drive, SD card, Blu-ray, and solid-state drive (SSD).

**Hint:** Magnetic storage involves spinning platters and uses electromagnets to read/write data. Optical storage involves lasers that create and read pits and lands on the surface of a disk. Solid-state storage uses NAND or NOR technology with transistors to store data.

---

### **Task 2: Comparing Secondary Storage**

(a) Explain why a computer needs secondary storage. (b) Compare three key differences between solid-state and magnetic storage in terms of speed, durability, and cost.

**Hint:** Secondary storage retains data permanently and provides long-term storage, as opposed to the temporary nature of primary storage. For part (b), consider that magnetic storage (e.g., HDD) is slower, less durable, but cheaper, while solid-state storage (e.g., SSD) is faster, more durable, and more expensive.

---

### **Task 3: Explain Secondary Storage Devices**

Match the examples of secondary storage devices (hard disk drive, CD, Blu-ray, USB drive, and solid-state drive) to their respective operations (magnetic, optical, or solid-state). Provide reasons for your choices.

**Hint:** Think about how the devices operate: Magnetic storage uses spinning platters and electromagnets (e.g., HDD). Optical storage uses lasers to create pits and lands (e.g., CDs and Blu-rays). Solid-state storage uses NAND gates and transistors (e.g., USB drives and SSDs).

---

### **Task 4: Analyze Virtual Memory**

(a) Describe what is meant by virtual memory. (b) Explain how virtual memory is created and used. (c) Describe a situation where virtual memory is necessary.

**Hint:** Virtual memory is created when the RAM is full, and it uses a portion of secondary storage as temporary storage. Unused pages of data are swapped from RAM to virtual memory, freeing up space for active tasks. This is especially useful when a system is handling multiple programs and running out of physical RAM.

---

(b) Explain two advantages and one disadvantage of cloud storage compared to local storage.

**Hint:** Cloud storage refers to data that is stored remotely and accessed over the internet. For advantages, think of how it enables remote access, automatic backup, and reduced hardware costs. A disadvantage could be dependence on internet connectivity or potential security concerns.

---

### **Task 6: Extended Writing**

Write a short paragraph explaining the roles of RAM, ROM, and secondary storage in a computer system.

**Hint:** RAM is volatile and used for temporary storage of data being actively used or processed. ROM is non-volatile and stores essential boot-up instructions for the computer. Secondary storage provides long-term, non-volatile storage for data and programs.

### **Task 7: Case Study**

A company needs to choose between HDDs and SSDs for its storage needs. Explain the advantages and disadvantages of each type of storage and recommend the most suitable one based on their requirements.

**Hint:** HDDs offer larger storage capacities at a lower cost but are slower and have moving parts, making them less durable. SSDs are faster, more durable, and consume less power but are more expensive. Consider the company’s priorities, such as speed, portability, or cost.

**Hint:** When RAM is full, data that is not actively needed is moved to virtual memory on the hard drive, freeing up space in RAM for new tasks. This allows the computer to handle more processes without freezing or crashing.
